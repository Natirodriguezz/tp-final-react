# Trabajo Final React

## Descripción del proyecto

Este proyecto fue desarrollado utilizando React y Vite como trabajo final.

La aplicación permite visualizar información de personajes mediante una interfaz dinámica. Cuenta con componentes reutilizables, una barra de búsqueda y conexión con una API para obtener y mostrar información.

El objetivo del proyecto es aplicar conocimientos de desarrollo frontend, trabajando con componentes, estilos, organización de archivos y consumo de datos externos.

## Lenguaje de progrmacion, diseño y estructura

- React
- Vite
- JavaScript
- HTML5
- CSS3

## Funcionalidades

- Visualización de personajes.
- Búsqueda de personajes.
- Uso de componentes reutilizables.
- Consumo de una API.
- Diseño de interfaz con CSS.


## Cómo ejecutar el proyecto localmente

1. Descargar o clonar el repositorio en la computadora.

2. Abrir la terminal dentro de la carpeta del proyecto.

3. Entrar a la carpeta donde se encuentra la aplicación React:
cd tp-react-final

4. instalar la dependencia del proyecto escribiendo:
 npm install

 5. despues ponemos para ejecutar la aplicacion:
 npm run dev 

6. abrir el navegador con la terminal que te aparecera asi:
 http://localhost:5173/
